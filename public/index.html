Cara Menggunakan

Tidak perlu install apapun. Cukup buka di browser dan mulai eksplorasi.

1
Pilih Scene
Klik salah satu destinasi di atas. Scene akan langsung terbuka di browser Anda.
2
Bergerak & Menjelajah
Desktop: WASD untuk bergerak, Mouse untuk melihat.
Quest 3: Joystick + gerak kepala.
3
Identifikasi Geometri
Arahkan kursor ke objek berwarna. Tahan 1,5 detik → panel info geometri muncul.
4
Mode VR Penuh
Di Quest 3, klik ikon 🥽 di sudut kanan bawah untuk masuk ke mode VR imersif penuh.

Kompatibilitas

Berjalan di Semua Perangkat

Dibangun dengan WebXR — teknologi VR berbasis web yang tidak memerlukan instalasi apapun.

🥽
Meta Quest 3
Mode VR imersif penuh. Gunakan Meta Quest Browser dan controller bawaan.
✓ VR Penuh
💻
Desktop / Laptop
Mode 3D interaktif. WASD + mouse. Bisa di laptop spesifikasi rendah sekalipun.
✓ Mode 3D
📱
Smartphone
Mode 3D dengan gyroscope. Putar HP untuk melihat sekeliling. Touch untuk klik objek.
✓ Mode 3D
🎮
Pico 4
Akses via Pico Browser. WebXR otomatis aktif. Gunakan controller untuk navigasi.
✓ VR Penuh
